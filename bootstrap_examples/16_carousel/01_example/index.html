Carousel Example

The Bootstrap carousel is a flexible, responsive way that is used to add a slider to your webpage. It is very responsive and flexible enough to allow, images, iframes, videos, or any other type of content that you want to add.

The Carousel Plugin:

The carousel plugin is a component for cycling through element, like a carousel.
If you want to add plugin functionality individually, then you have to use carousel.js file or all at once (using "bootstrap.js" or "bootstrap.min.js").

Display Property

Following is a list of some display utilities:

.d-block display utility: This utility is used to set an element display property to block.

Syntax:

<span class="d-block bg-primary">Block</span>
.d-inline utility: This utility is used to set an element display property to inline.

Syntax:

<div class="d-inline bg-success">Inline</div>
<div class="d-inline bg-success">Inline</div>
.d-inline-block: This utility is used to set an element display property to inline block.

Syntax:

<div class="d-inline-block bg-warning">
  <h3>inline-block</h3>
  Boot that strap!
</div>
<div class="d-inline-block bg-warning">
  <h3>inline-block</h3>
  Strap that boot!
</div>

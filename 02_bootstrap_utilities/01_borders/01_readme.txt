Borders
Border utility is used to quickly change the border-style and border-radius of an element.
 It is mainly used for images, buttons, or any other element.

You can use the border classes to an element to remove all borders or some borders.

Syntax:
For no border:

<span class="border-0"></span>

To remove top-border:

<span class="border-top-0"></span>

To remove right-border:

<span class="border-right-0"></span>

To remove bottom-border:

<span class="border-bottom-0"></span>

To remove left-border:


<span class="border-left-0"></span>
